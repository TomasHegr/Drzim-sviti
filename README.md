Držím-svítí
===========================

Účel/Zadání/Funkce
-----------------------
Při držení modrého tlačítka na STM8, začne svítít LED dioda na STM8. Když tlačítko pustím, LED dioda zhasne.


Schema zapojení
-----------------------

![schema zapojení](/docs/IMG_6441.jpeg)

Popis funkce
-----------------------

1. Stisknu a držím modré tlačítko na STM8
2. LED dioda svítí
3. Tlačítko pustím, LED dioda zhasne
